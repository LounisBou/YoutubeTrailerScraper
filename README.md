# YoutubeTrailerScraper 

YoutubeTrailerScraper is a python package that scan medias database folders to find movies and tvshows without trailers, search on youtube, and download it.

It is compose of the following specialized classes :

MovieScanner : Scan movies folders for movie and detect the one without trailers.
TVShowScanner : Scan tvshows folders for tvshow and detect the one without trailers.
YoutubeSearchEngine : Make search on youtube to find trailers videos.
YoutubeDownloader : Download a Youtube video from an URL.
YoutubeTrailerScraper : Use previous class and define the workflow and strategy.
